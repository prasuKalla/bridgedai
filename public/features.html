Chatbot and Virtual Assistant Training

Training data for Chatbots using domain specific conversation data that simulates conversations between Humans and Chatbots; developing conversations capable of delivering on several smart-interfaces. Use cases: Chatbots and Virtual Assistant training

Sentiment Analysis Training

Identifying keywords to track sentiment including sentiment labels such as Useful (positive) or Expensive (negative) in user reviews, user feedback, chatbot conversations or social media including tweets. Use cases include E-commerce, Chatbots, Social media analysis

Audio/Video Transcription

Converting audio/video files into professional transcription. Use Cases: Conference Calls, Investor Meetings, and Presentations. Use Cases include conference calls, investor meetings, and presentations

Image/Document Transcription

Converting text from images and documents (any format) to machine readable text for improved results. Use cases include Bills and invoices, Financial statements Menus and fliers, Prescriptions and records

NER Tagging for NLP

Carrying out Named-entity recognition (NER) - information extraction that locates and classifies entities in unstructured text into pre-defined categories such as person names, organizations, locations and so on.

Content Moderation

Moderating text, images, and videos to evaluate material allowing systems to highlight or block sensitive material. Use cases: Social media (fake news, inappropriate material), E-commerce (user reviews and feedback), Chatbots, Virtual Assistants

E-commerce Categorization

Categorizing products for better search including product, brand, color, size or other forms of categorization useful to improve search results and identifying similar products. Use cases: E-commerce

Search Relevance and Recommendations

Product meta-tagging and analysis to improve search relevance, provide better recommendation and drive conversions, including up-selling and cross-selling. Use cases: E-commerce

Fake Merchandise Identification

Moderating products to check for fake merchandise and intellectual property violations in listings across sellers and products. Use cases: E-commerce

Semantic segmentation

Pixel Level Labeling of images for extremely detailed annotation. We do this using our own tool or a standard tool that you may own or prescribe to our team. Use cases include Autonomous Vehicles, Robotics, Medical Imaging and Agri-tech.

Bounding Boxes & Cuboids

We work with both 2D and 3D coordinates including class labelling for identification. Use cases include Autonomous Vehicles, E-commerce and Retail Intelligence.

Polygons

Annotation of objects with irregular shapes with meaningfully greater accuracy to provide clients with high quality training data sets. Use cases include Autonomous Vehicles, Robotics, Drones, Medical Imaging and Agri-tech

Facial Recognition

Plotting a series of points to map and store facial features mathematically. The technique can also be used to detect variation in shapes and count minute objects. Use cases include facial recognition for identification/security purposes

Line Annotation

Line annotation involves the drawing and delineation of pathways (medians, shoulders) to train vehicles perception models for lane detection. Use Cases include Autonomous Vehicles

Video Annotation

Video annotation involves the detection / tracing of objects frame by frame in a series of images allowing systems to follow trajectory and/or change in position. Use cases include Robotics, Drones

3D Point Cloud Annotation

Identifying objects in a 3D point cloud and drawing bounding cuboids around the specified objects. Use Cases include Autonomous Vehicles

Optical Character Recognition

Detecting handwritten text from a source (PDF or image files) and converting it to machine readable text. Use Cases include Health-tech, E-commerce, Retail Intelligence
